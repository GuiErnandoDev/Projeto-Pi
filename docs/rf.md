# Requisitos Funcionais (RF)



*### RF01 — Login de usuário
O sistema deve permitir o login individualizado de clientes com e-mail e senha.

### RF02 — Alteração de senha
O sistema deve permitir que o cliente altere sua senha de acesso.

### RF03 — Recuperação de senha
O sistema deve disponibilizar uma função de recuperação de senha ("Esqueci minha senha").

## RF04 -Edição de informações
O sistema deve permitir a edição de dados básicos do perfil do cliente.

## RF05 -Aba de informação do cliente
O sistema deve exibir uma "Aba de exibição das informações do cliente.

## RF06 - Dowload de faturas 
O sistema deve listar as faturas do cliente e permitir o download delas em formato PDF.

## RF07 - Dashboard gráfico com consumo de energia
O sistema deve exibir um dashboard gráfico com o histórico de consumo de energia do cliente.

## RF08 - Dashboard gráfico de economina do cliente
O sistema deve exibir um dashboard destacando a economia financeira (em R$) gerada para o cliente.

## RF09 - Filtrar dashboard por mes/ano
O sistema deve permitir filtrar os dados dos dashboards por mês/ano.
